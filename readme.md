# react_chick

## Concept
It will start the development of react.
I made for my own.

## Using

```
> cd this/path
> npm install
```


add src/js directory

```
> gulp build
```

add dev/index.html


```
gulp server
```

browse 0.0.0.0:8080

```
gulp release
```
